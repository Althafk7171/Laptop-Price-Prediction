# Images

This folder contains visualization screenshots and model result images from the Laptop Price Prediction project.

## Files

- `price_distribution.png` - Distribution of laptop prices in the dataset. This helps understand the spread and skewness of the target variable.
- `company_distribution.png` - Brand/company distribution of laptops. This shows how many laptop records are available for each company.
- `company_price_comparison.png` - Company-wise laptop price comparison. This helps compare price differences across laptop brands.
- `model_comparison.png` - Comparison of machine learning models based on performance metrics such as R² score. This helps identify the best-performing model.
- `random_forest_result.png` - Final Random Forest Regressor result. This shows the best model performance used for laptop price prediction.
